# Task 1: Push Button Counter

## Objective
To count the number of times a push button is pressed and display the count on Serial Monitor / LCD.

## Components Used
- Arduino UNO / ESP32
- Push Button
- 10k Resistor
- Breadboard
- Jumper Wires

## Working
Each button press increases the counter value by 1 and displays it on Serial Monitor.

## Output
Serial Monitor displays the count.
